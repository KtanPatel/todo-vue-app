---------
Setup in Local
---------
### Requirement
- Nodejs LTS
- bun (`npm i -g bun`)

### Clone repo
`git clone https://github.com/KtanPatel/todo-vue-app.git`

### Install dependencies
`bun install`

### Run
`bun dev`
